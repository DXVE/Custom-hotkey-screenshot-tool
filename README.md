## 📌 工具简介
这是一个基于Python开发的桌面截图工具，拥有图形用户界面（GUI）。核心功能是允许用户自定义全局热键来触发系统截图，同时支持将程序最小化到系统托盘后台运行。

## ✨ 主要功能
1.  **自定义全局热键**：可自由设置截图和退出程序的热键组合。
2.  **系统托盘**：窗口最小化后，程序会隐藏到系统托盘，不影响前台操作。
3.  **图形化界面**：提供图形按钮进行手动截图和打开截图保存目录。
4.  **热键开关**：可临时禁用/启用截图热键功能。
5.  **操作日志**：界面内集成日志区域，实时显示热键注册、截图成功/失败等状态信息。

## 📦 安装与运行依赖
运行本脚本需要预先安装以下Python库：
- `tkinter` (通常随Python标准库安装)
- `keyboard`
- `pystray`
- `Pillow` (PIL)

```bash
pip install keyboard pystray Pillow
```

## 🚀 使用说明
1.  **启动程序**：运行脚本后，将显示主程序窗口；点击最小化时，程序会隐藏到系统托盘。
2.  **主界面概览**：
    - **热键设置区域**：可修改“截图热键”和“退出热键”。点击“应用”使新设置生效，点击“恢复默认”重置为初始热键。
        - 格式示例：`f4`, `ctrl+shift+q`, `alt+f12`
    - **按钮区域**：包含“手动截图”、“打开截图目录”和“退出”按钮。
    - **功能开关**：“启用截图（热键）”复选框，用于开启或关闭热键截图功能。
    - **状态显示**：显示程序当前运行状态（如“运行中”、“就绪”）。
    - **日志区域**：显示所有操作记录和状态信息。

3.  **系统托盘**：
    - 当主窗口最小化时，程序会自动隐藏并创建系统托盘图标。
    - 右键点击托盘图标，会显示以下菜单：
        - **手动截图**：执行截图。
        - **打开截图文件夹**：打开系统默认的截图保存目录。
        - **显示窗口**：恢复显示主程序窗口。
        - **退出程序**：完全退出应用程序。
4.  **截图保存**：工具通过模拟按下 `Win + PrintScreen` 键来触发系统原生截图功能。截图默认保存在系统的 `~\Pictures\Screenshots\` 目录下。

## 💡 示例
- 默认设置下，按 `F4` 键即可截图。
- 按 `Ctrl + Shift + Q` 可退出程序。

## ⚠️ 注意事项
- 本工具本质上是一个“热键触发器”，其截图功能依赖于Windows操作系统自身的截图机制（`Win + PrintScreen`）。
- 请确保设置的热键组合与其他应用程序不冲突。
- 退出程序时，建议通过托盘菜单的“退出程序”或主窗口的“退出”按钮进行操作，以确保正确清理后台监听线程。
